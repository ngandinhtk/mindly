# Mindly

Mindly is a minimal, pastel-style web application built with React for the frontend and Node.js with Express for the backend. This project aims to provide a simple and elegant user interface while maintaining a clean and efficient backend.

## Project Structure

```
mindly
├── client
│   ├── src
│   │   ├── components
│   │   ├── styles
│   │   └── index.tsx
│   ├── package.json
│   └── tsconfig.json
└── server
    ├── src
    │   ├── app.ts
    │   ├── routes
    │   └── controllers
    ├── package.json
    └── tsconfig.json
```

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd mindly
   ```

2. Install dependencies for the client:
   ```
   cd client
   npm install
   ```

3. Install dependencies for the server:
   ```
   cd server
   npm install
   ```

### Running the Application

1. Start the server:
   ```
   cd server
   npm start
   ```

2. Start the client:
   ```
   cd client
   npm start
   ```

The application should now be running on `http://localhost:3000`.
